# 📈 Прогресс разработки ИИ-Ассистента (Interview Assistant Progress)

Этот файл отслеживает текущее состояние, выполненные этапы и бэклог задач по развитию ИИ-Ассистента для технических собеседований.

---

## 🛠️ Текущий статус

- **Аудио-захват:** Реализован раздельный 2-канальный фоновый захват: микрофон (Канал 0, `pyaudio`) и системный звук (Канал 1, `BlackHole`).
- **Автоматический круговой буфер (Ring Buffer):** Звук пишется постоянно в кольцевой буфер глубиной 90 секунд. Больше нет необходимости вручную переключать старт/стоп записи.
- **Локальный STT с VAD:** Настроена модель `faster_whisper` (`small` на CPU) с фильтром тишины (`vad_filter=True`), отсекающим фоновый шум и предупреждающим галлюцинации.
- **Аппаратная диаризация:** Реализована раздельная транскрибация каналов со слиянием реплик по времени начала (`start`). Текст размечается тегами `[Candidate]` и `[Interviewer]`.
- **LLM-интеграция:** Подключена модель `gemini-2.5-flash` через Vertex AI (`litellm`). Системный промпт оптимизирован под диаризованный диалог для поиска последнего активного вопроса.
- **Вывод информации (Премиальный HUD):** Создан полностью переработанный полупрозрачный оверлей с эффектом Glassmorphic (размытие, скругленные углы 12px, мягкие тени `QGraphicsDropShadowEffect`).
- **Управление окном (Drag & Controls):** Добавлен верхний заголовок (Title Bar), за который можно свободно перетаскивать безрамочное окно мышкой. Встроены стандартные кнопки управления: сворачивание программы в док (`—`) и быстрое закрытие всего фонового процесса (`✕`).
- **Динамический размер окна:** Высота окна плавно анимируется при помощи `QPropertyAnimation` в диапазоне от 240px до 600px, адаптируясь под объем приходящего текста.
- **Защита от обрезания (Anti-clipping):** Добавлен встроенный `QScrollArea` с кастомным ультратонким 6px скроллбаром для удобного чтения длинных ответов.
- **Умное автоскрытие:** Окно автоматически скрывается с экрана в режиме ожидания (при нажатии `M` или статусе "Ready") и мгновенно появляется при запуске записи или генерации.
- **Стриминг:** Внедрен потоковый вывод ответов (`stream=True`) с инкрементальным парсингом на `TL;DR` и `Script`.
- **Интерактивное изменение размера:** Реализовано ручное перетаскивание границ и углов безрамочного окна для свободного изменения размера с комфортной чувствительной зоной 25px и авто-отключением анимации при ручном ресайзе.
- **Модульная архитектура бэкэндов:** Выделены 4 изолированных бэкэнда (Direct, Search RAG, Playbook Agent и Dialogflow CX) в отдельные файлы пакета `pipelines/`.
- **Очистка репозитория:** Неиспользуемые вспомогательные скрипты перемещены в директорию `archive/`.

## 📝 История версий (Changelog)

### Версия 1.1 (Текущая)

- Проведен модульный рефакторинг LLM-пайплайнов: логика каждого бэкэнда (Direct, Search RAG, Playbook Agent и Dialogflow CX) разделена на независимые модули внутри пакета `pipelines/`.

- Добавлена очистка и перенос неиспользуемых скриптов в папку `archive`.
- Переписан вызов Vertex AI на стриминг токенов в реальном времени.
- Добавлен инкрементальный парсер `_parse_incremental` в `main.py`.
- Добавлен оффлайн-режим загрузки Whisper (`local_files_only=True`), чтобы избежать зависаний при проверке обновлений на huggingface.co.
